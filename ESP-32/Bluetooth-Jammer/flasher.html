How it works

Arduino requires compiling C++ code — browsers can't do that. MicroPython is a Python interpreter that runs on the ESP32. Since Python code doesn't need compiling, we can write and send it directly from the browser over USB, making the whole thing work without any local tools.
Go to micropython.org/download/ESP32_GENERIC, click "Install" on the latest firmware — it uses the same Web Serial API and takes about 30 seconds. You only do this once.
Use a data-capable USB cable (not charge-only). Install the CP2102 or CH340 driver for your OS. Close any other apps using the serial port.
